Germline variant-calling pipeline

Read this page from top to bottom. Each stage shows the variables it uses (with short explanations), how to run it, and the full script in an editable window. Change any variable — every unlocked script and run command updates immediately. Accepts .fastq and .fastq.gz inputs.

Overview

Sequencing data (FASTQ → BAM → VCF) and array data (IDAT → PLINK → VCF) both converge at VCF for annotation and reporting. This repo implements the sequencing / clinical panel path.

Both pathways converge at VCF. This walkthrough covers the sequencing branch (left). The array branch (right) is documented but not yet implemented.
